---CLASSES BOOTSTRAP PARA MENU DE NAVEGAÇÃO---

navbar -> formatação base para barra de NAVEGAÇÃO

navbar-expand-lg -> expande/exibe todos os itens da barra de navegação quando a tela está cheia (lg).

navbar-light -> cor da letra da barra de navegação

bg-light -> cor do background da barra de navegação (se aplica em vários contextos)

navbar-brand -> formatação de logo na barra de navegação

navbar-toggler -> formatação do botão de menu

navbar-toggler-icon -> inserção do ícone do menu

collapse navbar-collapse -> unir os itens da div quando a tela for pequena

navbar-nav -> formatação padrão da barra de navegação (<ul>)

nav-item -> formatação do item (<li>) da barra de navegação

nav-link -> formatação dos links (<a>) da barra de navegação

dropdown -> pega a posição relativa do item para a expansão do dropdown

dropdown-toggle -> insere o ícone de seta para baixo no item para indicar que é um dropdown

dropdown-menu -> formatação da área do submenu

dropdown-item -> formatação dos itens do submenu

dropdown-divider -> linha divisória em um submenu


---JAVASCRIPT---

data-toggle = "collapse" -> funcionalidade do botão de menu (expandir e unir os itens do menu quando a tela estiver pequena)

data-target="#Conteúdo" -> referencia a ação do botão a uma tag que tenha o id "Conteudo"

data-toggle="dropdown" -> funcionalidade para o dropdown expandir ou unir os seus itens
